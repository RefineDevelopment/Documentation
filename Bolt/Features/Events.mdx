Events allow you to host and manage events on your server,
They allow your players to host events, that allow your ranked players to have fun.

### Commands

`<>` = Required `[]` = Optional

## ğŸ‰ Event Commands

Command                   | Permission        | Description                                    
--------------------------|-------------------|---------------------------------------------
`/event help`             | bolt.event.admin  | Displays the help message                   
`/event host <type>`      | bolt.event.admin  | Host a tournament                           
`/event info`             | bolt.event.admin  | Displays information about the current event
`/event join`             | bolt.event.admin  | Join the current event                      
`/event leave`            | bolt.event.admin  | Leave the current event                     
`/event forcestart`       | bolt.event.admin  | Force start the current event               
`/event cancel`           | bolt.event.admin  | Cancel the current event

### Creating an Event Arena

To create an event arena, you need to use the command,
/event arena create [Event Name],
Then go through the setup process, and you will have an event arena.
