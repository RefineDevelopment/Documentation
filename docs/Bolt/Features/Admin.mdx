# Admin

These commands are mainly used for certain things, such as giving yourself certain kit items, reset a user's profile etc. 

| Command                   | Permission                | Description                                    |
| ------------------------- | ------------------------- | ---------------------------------------------- |
| `/bolt reload`            | `bolt.admin.reload`       | Reload all configuration files                 |
| `/bolt save`              | `bolt.admin.save`         | Save all configuration files                   |
| `/bolt refill`            | `bolt.admin.refill`       | Refill your inventory with potions or soup     |
| `/bolt setLobby`          | `bolt.admin.setup`        | Set the lobby to the player’s current location |
| `/bolt goldenHead`        | `bolt.admin.setup`        | Receive a pre-made Golden Head                 |
| `/bolt fireball`          | `bolt.admin.setup`        | Receive a pre-made Fireball                    |
| `/bolt updateData`        | `bolt.admin.leaderboard`  | Save and update all leaderboards               |
| `/bolt reset`             | `bolt.admin.reset`        | Reset Bolt’s database and collections          |
| `/bolt resetStats <name>` | `bolt.admin.resetprofile` | Reset a player’s profile                       |
