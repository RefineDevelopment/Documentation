## Example Usage

Below are examples of how to use the core Carbon APIs.

```java
// Packet API - Handle packets and movement efficiently
PacketAPI packetAPI = PacketAPI.instance();
packetAPI.registerPacketHandler(plugin, new IPacketHandler() {
    @Override
    public <T> void handle(Player player, IPacketWrapper<T> packet, boolean fromClient) {
        // Handle packet
    }
});

// Chunk API - Lightweight chunk snapshots
IChunkAPI chunkAPI = IChunkAPI.instance();
ICarbonChunkSnapshot<?> snapshot = chunkAPI.takeSnapshot(chunk);

// Restore snapshot later (automatically handles refreshing)
chunkAPI.restoreSnapshot(chunk, snapshot);

// Combat API - Manage knockback profiles
CombatAPI combatAPI = CombatAPI.instance();
ICombatProfile profile = combatAPI.getProfile("default");

// Apply profile to player
combatAPI.setPlayerProfile(player, profile);

// Modify values dynamically
Collection<ICombatValue<?>> values = combatAPI.getKnockbackValues(profile);
for (ICombatValue<?> value : values) {
    // Check for specific value ID (e.g., "horizontal")
    if (value.getValue() instanceof Double && value.getID().equalsIgnoreCase("horizontal")) {
        ICombatValue<Double> doubleValue = (ICombatValue<Double>) value;
        // Apply new value to the profile instance
        doubleValue.setValue(profile, 0.45D); 
        break;
    }
}
```
