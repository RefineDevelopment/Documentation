# Citizens Fork

This specific fork of Citizens adds internal support for double-hit fixes and hooks directly into the Carbon combat system. We recommend using this fork over any other version to ensure full compatibility with Carbon 5.0's features.

### Information

:::tip
If you need to use another version and face issues, please make a ticket on our Discord. This fork is specifically optimized for Carbon 5.0.
:::
