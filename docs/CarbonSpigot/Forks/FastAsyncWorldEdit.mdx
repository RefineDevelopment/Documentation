# FAWE Legacy

The standard FAWE version may not work with Java 11+. Carbon features native support for **FAWE-Legacy**, which is compatible with Java 17+ and Java 21. It is highly recommended to use this version alongside Java 21 for optimal performance.

### Requirements

:::important
You must use **WorldEdit 6.1.7** with this FAWE version to ensure full compatibility.
:::
