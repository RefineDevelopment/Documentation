# Permissions

Below is a list of default permissions available in CarbonSpigot.

| Command | Permission | Description |
| :--- | :--- | :--- |
| `/carbon` | `carbonspigot.help` | Main help command |
| `/carbon settings` | `carbonspigot.settings` | Access to Carbon settings |
| `/carbon menu` | `carbonspigot.menu` | Opens the Carbon menu |
| `/killall` | `carbon.clearentities` | Clears all entities in the world |
| `/setslots` | `carbon.setslots` | Change server player slots |
| `/hide` | `carbon.hide` | Toggle player visibility |
| `/plugins` | `carbon.plugins` | Custom optimized plugins command |
| `/knockback` | `carbon.Knockback` | Knockback profile management |
| `/potion` | `carbon.potion` | Potion effect management |
